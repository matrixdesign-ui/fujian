# Fujian Concrete Works Website

This is the official website for Fujian Concrete Works, a construction materials and services company based in Nairobi, Kenya.

## Features

- Responsive design for all device sizes
- Modern UI with Bootstrap 5
- Interactive elements and animations
- SEO optimized
- Accessible navigation
- Contact form with validation
- Cookie consent banner

## Deployment

This website is designed to be deployed on GitHub Pages with a custom domain.

## Custom Domain

The site is configured to use the custom domain: www.fujianworks.com

To set up the custom domain:
1. Add the domain to your GitHub repository settings
2. Configure DNS records with your domain provider:
   - CNAME record pointing to `username.github.io` (where username is your GitHub username)
   - A records for the apex domain if needed

## GitHub Pages Setup

1. Go to your repository settings
2. Scroll down to the "Pages" section
3. Select the branch to deploy from (usually `main`)
4. Save the settings
5. The site will be available at https://username.github.io/repository-name

## Local Development

Simply open `index.html` in a web browser to view the site locally.
